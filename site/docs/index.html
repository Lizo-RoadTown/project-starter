Plain-language docs for project-starter. Built for people who are new to coding, new to Claude Code, or just want to understand what this template actually does before they use it.

What you'll have when you're done

By the end of the walkthrough you'll have:

  • A new folder on your computer with a clean project structure — special files that tell Claude Code how your project is set up.
  • A copy of that folder on GitHub — a website that hosts code and tracks every change you ever make.
  • Claude Code open and connected to your project, ready to help you build features by typing what you want in plain English.

What this is not for

Just so it's clear up front:

  • This isn't an AI tutorial — it doesn't teach you to use Claude Code itself beyond the first conversation. For that, see Claude Code's own docs.
  • This isn't a coding course — it gives you a folder to code in, but you'll still need to know (or ask Claude) what to build.
  • This isn't required to use Claude Code. You can use Claude Code on any project. This starter just gives every new project a consistent, well-tuned starting point.
